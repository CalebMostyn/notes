# vim cheatsheet

## Motions / Shortcuts
- f[x] : jump to before next occurance of char x
- t[x] : jump to after next occurance of char x
- F[x] : jump to before previous occurance of char x
- T[x] : jump to after previous occurance of char x
- H : move to top of screen
- M : move to middle of screen
- L : move to bottom of screen
- U : highlight to uppercase
- u : highlight to lowercase
- Ctrl+a : increment number under cursor
- Ctrl+x : decrement number under cursor
## Commands
- :w : write file
- :wq : write file and quit
- :wa : write all files
- :sort : sorts highlighted content
